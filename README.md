# country-
CountryData is an exploratory data analysis project that examines global country statistics through data cleaning, visualization, and statistical analysis to uncover meaningful insights about development indicators.   


CountryData:Data Analysis of Global Country Indicators

 Project Overview:
This project explores relationships between social, economic, environmental, and demographic indicators across countries. The goal is to clean the dataset, perform exploratory data analysis (EDA), and uncover meaningful insights using data visualization.
This project focuses on understanding how different country-level indicators relate to one another rather than building machine learning models.

Objectives:
Clean and prepare the dataset for analysis.
Explore relationships between key country indicators.
Visualize trends and correlations using charts.
Generate insights that can support data-driven decision-making.

Data Cleaning:
The following preprocessing steps were performed:
Removed duplicate records.
Handled missing (null) values.
Filled missing values where appropriate.
Detected and removed outliers.
Checked data quality before analysis.

 Exploratory Data Analysis:
The project investigates the following relationships:
GDP vs. Infant Mortality
GDP vs. Male Life Expectancy
GDP vs. Female Life Expectancy
GDP vs. CO₂ Emissions Rate
Female Education vs. Fertility Rate
Urban Population Growth vs. Internet Users
International Tourists vs. GDP
Surface Area vs. Threatened Species
Unemployment Rate vs. Homicide Rate

Visualizations:
The analysis includes:
Scatter plots
Histograms
Bar charts
Bar plots
Visualizations were created using Matplotlib,Seaborn and Plotly.

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Plotly
Seaborn

Key Takeaways:
Some relationships showed clear positive or negative trends, while others exhibited weak or no correlation. These findings demonstrate how exploratory data analysis can reveal patterns, identify anomalies, and provide a deeper understanding of global development indicators.
